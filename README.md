# README

Mahisti Clicker is an incremental clicker game inspired by [A Taste of Gold and Iron](https://www.alexandrarowland.net/a-taste-of-gold-and-iron) written by [Alexandra Rowland](https://www.alexandrarowland.net/).

## Change Log
